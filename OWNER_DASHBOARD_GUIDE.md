## 📊 Analytics - Phân Tích

### Truy Cập
Nhấn menu **"📊 Phân Tích"** trên sidebar

### 5 Loại Phân Tích Chính

#### 1️⃣ Revenue (Doanh Thu)
```
Hiển thị:
- Tổng doanh thu từ tất cả nguồn
- Doanh thu từ lịch hẹn
- Doanh thu từ đơn hàng online
- Doanh thu trung bình mỗi lịch hẹn
- Tỷ lệ phối hợp (%)

Công dụng:
→ Theo dõi tình hình kinh doanh
→ So sánh nguồn doanh thu
→ Dự báo tăng trưởng
```

#### 2️⃣ Customer (Khách Hàng)
```
Hiển thị:
- Tổng khách hàng đã đăng ký
- Số khách hoạt động
- Số khách trung thành (≥5 lần mua)
- Số khách mới (30 ngày)
- Tỷ lệ giữ chân (%)
- Giá trị vòng đời trung bình (CLV)

Công dụng:
→ Quản lý chất lượng khách hàng
→ Đánh giá hiệu quả marketing
→ Lập kế hoạch giữ chân khách
```

#### 3️⃣ Staff Performance (Hiệu Suất Nhân Viên)
```
Hiển thị (cho mỗi nhân viên):
- Tên và số điện thoại
- Số lịch đã thực hiện
- Tổng doanh thu tạo ra
- Rating từ khách hàng (1-5 sao)
- Trạng thái (Hoạt động/Nghỉ)

Công dụng:
→ Đánh giá hiệu suất từng thành viên
→ Lập danh sách công nhân xuất sắc
→ Cải thiện dịch vụ
```

#### 4️⃣ Package Analytics (Gói Dịch Vụ)
```
Hiển thị:
- Tên gói dịch vụ
- Số lần mua (Phổ biến)
- Giá bán
- Tổng doanh thu từ gói

Công dụng:
→ Xác định gói dịch vụ bán chạy
→ Tối ưu portfolio sản phẩm
→ Lập kế hoạch khuyến mãi
```

#### 5️⃣ Product Analytics (Sản Phẩm)
```
Hiển thị:
- Tên sản phẩm
- Tồn kho
- Số bán ra
- Giá bán

Công dụng:
→ Quản lý bán hàng lẻ
→ Cảnh báo hàng sắp hết
→ Tối ưu danh mục sản phẩm
```

## 🎯 Marketing - Kinh Doanh

### Truy Cập
Nhấn menu **"🎯 Marketing"** trên sidebar

### Tab 1: Khuyến Mãi (Promotions)

**Tạo Khuyến Mãi Mới:**
1. Nhấn nút **"+ Thêm Khuyến Mãi"**
2. Điền thông tin:
   - **Tên khuyến mãi**: VD "Giảm 10% gói massage"
   - **Loại**: Chọn từ dropdown (Discount, Loyalty, Flash Sale, Referral)
   - **Giá trị**: Nhập con số (ví dụ: 10, 1, 50)
   - **Hết hạn**: Chọn ngày kết thúc khuyến mãi
3. Nhấn **"Lưu"**

**Ví Dụ:**
- Discount 10%: Giảm 10% cho gói dịch vụ
- Loyalty 1: Mua 5 lần tặng 1 lần
- Flash Sale 30: Giảm giá 30% trong hôm

### Tab 2: Chiến Dịch (Campaigns)

Hiển thị các chiến dịch marketing:
- Email chào hàng
- SMS nhắc lịch hẹn
- Thông báo khuyến mãi

**Tính năng:**
- Xem trạng thái (Đã gửi, Sắp tới)
- Đếm số người nhận
- Chi tiết chiến dịch

### Tab 3: Loyalty Program (Chương Trình Trung Thành)

**Cài Đặt Điểm Thưởng:**
```
Điểm/100,000 VND: 1 điểm (mặc định)
Giá Trị 1 Điểm: 10,000 VND (mặc định)

Ví dụ:
- Khách mua gói 500,000 VND → Được 5 điểm
- Tích 50 điểm → Đổi được 500,000 VND giảm giá
```

**Cấp Độ Thành Viên:**
```
🥉 Bronze (Đồng):  Từ 100 điểm
🥈 Silver (Bạc):   Từ 500 điểm
🥇 Gold (Vàng):    Từ 1,000 điểm

Lợi ích:
- Nhìn được số khách mỗi cấp
- Cung cấp ưu đãi khác nhau
- Khuyến khích khách tiêu dùng thêm
```

## 💳 Financial - Tài Chính

### Truy Cập
Nhấn menu **"💳 Tài Chính"** trên sidebar

### Tab 1: Tổng Quan (Overview)

**4 Chỉ Số Chính:**
```
📈 Tổng Doanh Thu
   Tính từ: Lịch hẹn + Đơn hàng online
   
📉 Tổng Chi Phí
   Tính từ: Lương, thuê, vật tư...
   
💵 Lợi Nhuận
   = Doanh Thu - Chi Phí
   
💳 Tiền Mặt
   Tổng trong tất cả tài khoản
```

**Bảng Dòng Tiền:**
```
Doanh Thu (brutto)      = ₫X
- Chi Phí              = ₫Y
= Lợi Nhuận Trước Thuế  = ₫Z
- Thuế TNDN            = ₫A
- VAT                  = ₫B
= Lợi Nhuận Sau Thuế    = ₫(Z-A-B)
```

### Tab 2: Hóa Đơn (Invoices)

**Xem Lịch Sử:**
```
Cột:  Số HĐ | Ngày | Khách | Tiền | Loại | Trạng Thái | Hành Động

Loại: Lịch Hẹn / Đơn Online
Trạng Thái: Đã Thanh Toán / Chờ TT
```

**Tạo Hóa Đơn:**
- Tính năng sẽ cập nhật sớm
- Hiện tại xem lịch sử tự động

### Tab 3: Chi Phí (Expenses)

**Thêm Chi Phí:**
1. Nhấn **"+ Thêm Chi Phí"**
2. Chọn danh mục (Lương, Thuê, Vật tư)
3. Nhập số tiền
4. Thêm ghi chú
5. Lưu

**Biểu Đồ Chi Phí:**
```
Hiển thị % chi phí mỗi danh mục
Giúp tối ưu hóa chi tiêu
```

### Tab 4: Phương Thức Thanh Toán

**4 Kênh Thanh Toán:**
```
💵 Tiền Mặt     - Thanh toán trực tiếp
🏦 Chuyển Khoản  - Chuyển vào tài khoản ngân hàng
📱 MoMo          - Ví điện tử
💳 Thẻ Tín Dụng  - Thanh toán bằng thẻ
```

Hiển thị:
- Tổng tiền mỗi kênh
- Tỷ lệ (%) so với tổng

### Tab 5: Quản Lý Thuế

**Cài Đặt Thuế:**
```
Năm Tài Chính: 2024
Thuế TNDN: 10%
VAT: 8%
Áp dụng Thuế TNDN: ☑ Có
```

**Tính Toán Tự Động:**
```
Doanh Thu Năm           = ₫X
- Chi Phí              = ₫Y
= Lợi Nhuận Trước Thuế  = ₫Z

Thuế TNDN (10%)        = ₫Z × 10%
VAT (8%)               = ₫X × 8%
Tổng Thuế              = ₫(Z×10% + X×8%)

Lợi Nhuận Sau Thuế      = ₫Z - Tổng Thuế
```

## 👥 HR - Nhân Sự

### Truy Cập
Nhấn menu **"👥 Nhân Sự"** trên sidebar

### Tab 1: Nhân Viên (Staff)

**Thêm Nhân Viên:**
1. Nhấn **"+ Thêm Nhân Viên"**
2. Điền:
   - Họ và tên
   - Số điện thoại
   - Vị trí (Therapist, Receptionist, Manager, Assistant)
   - Lương cơ bản
3. Nhấn Lưu

**Quản Lý:**
- Xem danh sách nhân viên
- Xóa nhân viên
- Xem rating khách đánh giá
- Trạng thái (Hoạt Động/Nghỉ)

### Tab 2: Chấm Công (Attendance)

**Hiển Thị Tự Động:**
```
Tên | Điểm Danh | Tổng Ngày | Tỷ Lệ (%)
```

**Tính Toán:**
- Mặc định: 22 ngày làm việc/tháng
- Tỷ lệ = (Ngày đi / 22) × 100%

**Ý Nghĩa:**
- 100%: Không nghỉ
- 90%: Nghỉ 2-3 ngày
- <80%: Nhiều ngày vắng

### Tab 3: Lương (Payroll)

**Lịch Sử Thanh Toán:**
```
Tháng | Tổng Lương | Trạng Thái | Ngày TT
```

**Chi Tiết Lương (Mẫu):**
```
Lương Cơ Bản           = ₫6,000,000
+ Phụ Cấp              = ₫500,000
+ Thưởng               = ₫1,000,000
────────────────────────────────
= Lương Brutto         = ₫7,500,000

- Bảo Hiểm (8%)        = ₫600,000
- Thuế (5%)            = ₫435,000
────────────────────────────────
= Lương Ròng           = ₫6,465,000
```

**Cài Đặt:**
- Lương cơ bản mặc định
- Phụ cấp, thưởng
- Tỷ lệ bảo hiểm, thuế

### Tab 4: Đánh Giá (Performance)

**Chỉ Số Đánh Giá:**
```
Nhân Viên | Rating | Số Lịch | Lương | Chi Tiết
```

**Rating:**
- ⭐ 5: Xuất sắc
- ⭐ 4: Tốt
- ⭐ 3: Trung bình
- ⭐ 2: Cần cải thiện
- ⭐ 1: Chưa đạt

## 📦 Inventory - Kho Hàng

### Truy Cập
Nhấn menu **"📦 Kho Hàng"** trên sidebar

### Tab 1: Tổng Quan (Overview)

**4 Chỉ Số:**
```
📊 Tổng Sản Phẩm
   Tổng số hàng trong kho
   
⚠️ Hàng Cảnh Báo
   Sắp hết, cần nhập lại
   
✓ Hàng Tốt
   Đủ tồn kho
   
✗ Hết Hàng
   Hết stock, cần nhập gấp
```

**Cảnh Báo Tự Động:**
```
Nếu tồn kho < mục tiêu → Hiển thị cảnh báo
Ví dụ: Tinh dầu spa (tồn 3, mục tiêu 5)
```

### Tab 2: Danh Sách Sản Phẩm (Products)

**Cột Hiển Thị:**
```
Tên | SKU | Tồn Kho | Mục Tiêu | Vị Trí | Trạng Thái
```

**Trạng Thái:**
```
✓ Tốt      - Tồn kho >= mục tiêu
⚠ Cảnh báo - Tồn kho < mục tiêu
✗ Hết      - Tồn kho = 0
```

**Vị Trí Lưu Trữ:**
```
Kệ A1, Kệ B2, Kệ C1, ...
Giúp tìm hàng nhanh
```

### Tab 3: Lịch Sử Giao Dịch (Transactions)

**Thêm Nhập Hàng:**
1. Nhấn **"+ Nhập Hàng"**
2. Chọn:
   - Sản phẩm
   - Số lượng
   - Nhà cung cấp
   - Ghi chú (tùy chọn)
3. Nhấn Lưu

**Xem Lịch Sử:**
```
Ngày | Loại | SKU | Số Lượng | Từ/Đến | Ghi Chú

Loại:
- Nhập: Hàng nhập về
- Xuất: Hàng bán ra
- Điều Chỉnh: Kiểm kê, hư hỏng
```

### Tab 4: Nhà Cung Cấp (Suppliers)

**Danh Sách:**
```
Tên | Liên Hệ | Email | Trạng Thái
```

**Cách Dùng:**
- Chọn nhà cung cấp khi nhập hàng
- Quản lý danh sách các nhà cung cấp

## 💡 Best Practices

### 1. Analytics - Tối Ưu Kinh Doanh
```
Hàng ngày:
→ Kiểm tra doanh thu
→ Nếu có hóa đơn chưa thanh toán → Nhắc nhở khách

Hàng tuần:
→ Xem hiệu suất nhân viên
→ Gói dịch vụ nào bán chạy nhất
→ Lập kế hoạch khuyến mãi

Hàng tháng:
→ Phân tích khách mới/cũ
→ Tỷ lệ giữ chân khách
→ Dự báo doanh thu
```

### 2. Marketing - Phát Triển Kinh Doanh
```
Mỗi tháng:
→ Tạo 1-2 khuyến mãi hấp dẫn
→ Tính điểm loyalty cho khách trung thành
→ Lập campaign email/SMS

Mỗi quý:
→ Lên kế hoạch promotion lớn
→ Đánh giá hiệu quả loyalty program
→ Điều chỉnh cấp độ thành viên
```

### 3. Financial - Kiểm Soát Chi Phí
```
Mỗi tuần:
→ Kiểm tra chi phí chủ yếu
→ So sánh doanh thu/chi phí

Mỗi tháng:
→ Tính lương nhân viên
→ Kiểm tra hóa đơn chưa thanh toán
→ Sắp xếp lại chi phí

Mỗi năm:
→ Tính toán thuế
→ Lập báo cáo tài chính
→ Nộp hồ sơ khai thuế
```

### 4. HR - Quản Lý Nhân Lực
```
Mỗi ngày:
→ Kiểm tra chấm công
→ Theo dõi hiệu suất

Mỗi tháng:
→ Thanh toán lương
→ Đánh giá hiệu suất
→ Lập kế hoạch bonus

Mỗi quý:
→ Đánh giá tổng thể
→ Lên kế hoạch cải thiện
→ Tính thưởng/điều chỉnh lương
```

### 5. Inventory - Quản Lý Kho
```
Hàng ngày:
→ Kiểm tra cảnh báo hàng sắp hết
→ Nhập hàng khi cần

Hàng tuần:
→ Kiểm tra tồn kho chính xác
→ So sánh với bán ra

Hàng tháng:
→ Kiểm kê toàn bộ kho
→ Điều chỉnh số liệu
→ Lên kế hoạch nhập hàng
```
